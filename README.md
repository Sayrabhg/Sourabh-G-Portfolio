## ⚙️ Setup & Installation

1. Clone the repository:

```bash
git clone https://github.com/Sayrabhg/Sourabh-G-Portfolio.git
```

2. Navigate to the folder:

```bash
cd Sourabh-G-Portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```
